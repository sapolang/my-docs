## 🚀 快速开始

### 环境要求
- Node.js 16+
- pnpm (推荐) 或 npm

### 安装依赖
```bash
pnpm install
```

### 本地开发
```bash
pnpm run dev
```
访问 pnpm run dev

### 构建生产版本
```bash
pnpm run build
```

### 预览构建结果
```bash
pnpm run preview
```

## 📦 部署

### GitHub Pages (推荐)

1. Fork 本仓库
2. 在 GitHub 仓库设置中启用 Pages
3. 选择 \"GitHub Actions\" 作为源
4. 推送代码到 main 分支，自动触发部署

### 其他平台

- **Netlify**: 连接 GitHub 仓库，构建命令 `pnpm run build`，发布目录 `.vitepress/dist`
- **Vercel**: 导入项目，选择 VitePress 框架预设
- **自己的服务器**: 执行 `./deploy.sh` 脚本
